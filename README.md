# SalaryCalc Pro – CTC to In-Hand Salary Calculator

A modern React application that converts **CTC (Cost to Company)** into **accurate in-hand salary** using **India FY 2026–27 New Tax Regime**, featuring a clean UI and smooth animations.

---

## Features

- Real-time CTC → Net salary calculation  
- India FY 2026–27 new tax regime logic  
- Dark / Light mode toggle  
- Interactive sliders for salary components  
- Responsive pie-chart breakdown  
- Mobile-first, responsive UI  
- Smooth Framer Motion animations  

---

## Tech Stack

- **Frontend:** React 18  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Charts:** Recharts  
- **Icons:** Lucide React  

---

## How It Works

Input: CTC ₹12,00,000
Process: Tax + PF + HRA calculation
Output: ₹70,000/month (approx.)


## Tax Regime (FY 2026–27)

0–3L : 0%
3–7L : 5%
7–10L : 10%
10–12L : 15%
12–15L : 20%
15L+ : 30%
